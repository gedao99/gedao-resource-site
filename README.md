# 格道资源站

一个简洁的资源分享网站，支持三大分类：破解版APP、AI学习教程、副业资源。

## 功能特点

- 📱 三大资源分类展示
- 🔐 管理员后台（密码保护）
- 🎨 美观的响应式设计
- ⚡ 快速部署到 Vercel
- 🆓 完全免费使用

## 管理密码

默认管理密码：`gedao2024`

进入 `/admin` 页面进行管理。

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 部署到 Vercel

1. 将代码推送到 GitHub
2. 在 Vercel 导入项目
3. 一键部署完成

## 技术栈

- Next.js 14
- React 18
- TypeScript
- 静态导出（无需数据库）
